### 统一稿件状态码 <AuthorBadge text="武上博、刘嘉伟"/>

为了将「文章撰写」子模块和「稿件管理」子模块更好的配合，我和负责稿件管理子模块的刘嘉伟同学设计了一套稿件状态码，用于统一标识稿件状态。

| 稿件状态码 |       稿件状态      |
| :---: | :-------------: |
|  1001 |     草稿，尚未审核     |
|  2001 |  已提交审核，进入一级审核队列 |
|  2002 | 一级审核通过，进入二级审核队列 |
|  2003 | 二级审核通过，进入三级审核队列 |
|  2004 |  三级审核通过，进入待发布队列 |
|  3001 | 第三方平台已发布，稿件发布完成 |
|  4001 |  稿件时效性降低，进入归档队列 |
